# Partials
Partials opbygger de semantiske tags af enkelte komponenter.
De har lidt logik og begynder at danne UI-elementer med reel funktionalitet.

I denne løsning ligger de overordnede tags (header, nav, footer) i index.html og de enkelte partiale komponenter indsætter deres indhold direkte i DOM'en.

Eksempler:

* Header
* Navbar (Ul)
* Footer

## Placering:
```
/js/views/partials/
```
