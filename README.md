# Sgt. Prepper Frontend
Et frontend boilerplate til undervisning i opbygning af en webshop med ren HTML, Tailwind CSS og vanilla JavaScript. 

Projektet er struktureret efter MVC og Atomic Design og indeholder eksempel på routing, komponentopbygning og datahåndtering, så elever kan arbejde uden frameworks og stadig følge moderne arkitekturprincipper.

Projektet fungerer desuden som boilerplate til kommende code-along sessions.
